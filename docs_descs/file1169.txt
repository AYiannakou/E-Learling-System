В языке программирования java в стандартной библиотеке, есть множество различных способов хранения данных. Которые различаются по способу хранения тем самым различается различное поведение работы с коллекцией.

В этом видео уроке рассмотрена тема хранения данных в коллекции List на основе списка, рассматриваются две реализации интерфейса List:

ArrayList - Хранение данных на базе массивов
LinkedList - Хранение данных в виде двунаправленных связных списков.

В зависимости от задачи необходимо выбрать ту или иную реализацию интерфейса List.
