Ein wenig schwere Hausi dieses mal. Hier einige Tips:

Zum einlesen eines Strings vom Benutzer braucht hier die Scanner Variable und dann z.b. 
s = scan.next();

Dann zum schauen ob ein String einen Wert hat (bei If) geht leider kein "==", dafür müsst ihr wieder eine Methode benutzen. Beispiel:
if(s.equals("+")){
}

Wobei s der String ist!
