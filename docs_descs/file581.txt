Das Programm tauscht mit Hilfe einer Variable temp 2 Zeichen eines char Arrays.
Kopiere die main-Methode in eine neue Klasse. Erweitere das Programm so, dass am Ende Bleistift statt Beilstift im Array steht. Dafür sollen ebenfalls Zeichen im Array vertauscht werden. 
Nicht erlaubt sind direkte Zuweisungen von charactern wie: 
myArray[2] = 'e'; 

Aufgabe 3.1
schreibe ein neues Programm das ein String-Array (String[]) verwendet um deine Adressdaten zu speichern. Auf Index 0 wird der Vorname eingetragen, auf Index 1 der Nachname, 2: Straße, 3: Hausnummer, 4: Postleitzahl. Das Programm soll dann Die Kontaktdaten im Array auf der Konsole ausgeben. Dabei soll eine Zeile für den kompletten Namen und eine Zeile für die Anschrift verwendet werden.
