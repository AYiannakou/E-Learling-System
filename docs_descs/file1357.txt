Une société veut transmettre des données par téléphone, mais elle tient compte de ce que les lignes sont exposées au  piratage alors toutes les données doivent être encrypté avant d'être transmise au destinataire. Pour ce faire, vous aurez besoin d'un attribut de type String que vous nommerez chaine,  pour la saisie du mot que vous voulez encrypter. Après la saisie de la chaine, convertissez chaque lettre en sa valeur ASCI. Pour recevoir chacune de ces valeurs avant modification, déclarez un attribut de type int que vous nommerez valeur. Vous aurez besoin d'une boucle for pour accéder à chacune des lettres. Pour chaque valeur saisie effectuer le calcul suivant : ( ( valeur * 2 ) % chaine.length() ) . Convertissez ensuite la nouvelle valeur en String  avec la méthode  Integer.toString( valeur ). Cumulez ainsi chaque valeur dans un attribut de type String que vous nommerez chaineSecrete. La classe se termine en affichant la chaine encryptée.
