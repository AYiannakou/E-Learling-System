En esta ocación veremos un método para ordenar objetos de tipo List como un ArrayList o LinkedList, primero ordenaremos un ArrayList de Strings y luego uno de Personas. Para esto nos guiamos en el API de Java que es una gran ayuda a la hora de programar.
Vemos conceptos como la clase Collections, metodo sort, interface Comparable y el método compareTo.
